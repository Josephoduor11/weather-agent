# 🌤️ Weather AI Assistant

An intelligent weather assistant powered by LangChain, OpenRouter, and Groq's Llama 3.3 model. Get real-time weather conditions and forecasts through natural conversation.

## ✨ Features

- **Real-time Weather Data** - Get current weather conditions for any city worldwide
- **Weather Forecasts** - Access 5-day weather predictions
- **Natural Language Understanding** - Ask about weather naturally, just like talking to a friend
- **Conversation Memory** - The assistant remembers your conversation context
- **Tool Calling** - Intelligent use of weather tools based on your questions
- **Multi-Model Support** - Compatible with Groq, OpenRouter, and DeepSeek models

## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher
- OpenWeatherMap API key (free tier available)
- API key for your chosen LLM provider:
  - [Groq](https://console.groq.com) (free tier available)
  - [OpenRouter](https://openrouter.ai/keys) (free tier available)
